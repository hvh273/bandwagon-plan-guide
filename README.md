## 常见问题 FAQ

**Q：搬瓦工推送套餐（限量版）和常规套餐有什么区别？**

A：推送套餐是限量发售的特价方案，同等配置通常比常规套餐便宜 50%-70%。缺点是库存少、不稳定，随时可能缺货。常规套餐随时可买，但价格高得多。

**Q：限量版套餐买了可以换机房吗？**

A：部分可以，部分不行。传家宝系列（CN2 GIA-E 限量版）支持 14 个机房切换，THE PLAN 系列支持 18 个机房。MEGABOX-PRO、The DC6 Plan、MINICHICKEN 等套餐锁定指定机房，不支持迁移。购买前确认套餐说明。

**Q：推送套餐补货有规律吗，大概什么时候有货？**

A：没有固定规律。搬瓦工从不公布补货计划。历史上 MINICHICKEN 大约每两个月补一次，但这也不是官方确认的。最稳的方式就是订阅官方邮件通知。

**Q：抢不到限量版，常规套餐怎么选？**

A：常规套餐按预算选：入门就选 KVM 套餐（$49.99/年起），对速度有要求选 CN2 GIA-E（$169.99/年起），预算够的选香港 CN2 GIA（$89.99/月起）。

**Q：搬瓦工支持支付宝吗？**

A：支持。下单时选择 Alipay 即可，国内用户直接扫码支付，不需要信用卡。
